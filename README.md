# Beautiful Clock & Backgrounds 🕰️✨

A minimalist yet highly customizable Chrome extension that transforms your "New Tab" page into a beautiful digital clock with rotating high-quality backgrounds. Designed to be both functional and aesthetically pleasing, it helps you keep track of time while keeping your workspace fresh and inspiring.

## 🌟 Key Features

- **Sleek Digital Clock**: A large, easy-to-read digital clock with a live-updating date display.
- **Dynamic Backgrounds**: Automatically rotates through a gallery of beautiful, high-resolution landscapes and artistic images.
- **Total Customization**: 
    - **Fonts**: Choose from over 25+ premium Google Fonts for both the clock and the date.
    - **Colors**: Fully customizable color pickers for every text element.
    - **Positioning**: Shift the clock layout to the Top, Center, or Bottom of the page.
- **Personal Background Gallery**:
    - **Upload Local Images**: Add your own favorite photos directly from your computer.
    - **Image URLs**: Link directly to high-quality images from the web.
    - **Toggle Collection**: Easily switch between the default gallery and your personal collection.
- **Custom Text Overlays**: Add personal notes, quotes, or reminders with their own dedicated fonts and colors.
- **Keyboard Shortcuts**: Navigate through backgrounds instantly using your keyboard.

## 🛠️ How to Use

### Accessing Settings
Click the **Gear Icon (⚙️)** in the bottom center of the screen to open the settings panel. From here, you can customize every aspect of the extension.

### Settings Guide

1.  **Appearance**:
    - Use the dropdowns to change the font for the **Clock** and **Date**.
    - Click the color boxes to choose a custom color that matches your favorite background.
    - Change the **Layout Position** to move the clock if it's blocking a beautiful part of an image.
2.  **Custom Texts & Notes**:
    - Type a note in the input box, select its font and color, and click **"Add Text"**.
    - These will appear right below the clock, perfect for daily goals or mantras.
3.  **Backgrounds**:
    - **Change Interval**: Set how many minutes should pass before the background automatically rotates (1-60 mins).
    - **Use Local Backgrounds**: Check this to stop using the default gallery and start using your own collection.
    - **Add images**: Browse your computer for files or paste a direct image URL.

### Keyboard Shortcuts
- **Right Arrow (→)**: Manually skip to the next background.
- **Left Arrow (←)**: Go back to the previous background.

## 🚀 Installation (Developer Mode)

1.  Download or clone this repository.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Enable **"Developer mode"** toggle in the top right corner.
4.  Click **"Load unpacked"** and select the folder containing these files.
5.  Open a new tab and enjoy!

## 💻 Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Storage**: Chrome Storage API (`sync` for settings, `local` for large image data)
- **External Resources**: Google Fonts API, Picsum Photos
